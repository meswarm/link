# 系统查询技能

## 适用场景

- 用户询问"系统状态怎么样"、"报告一下系统信息"
- 用户需要综合性信息（时间 + 系统状态）
- 用户询问关于运行环境的问题

## 工作流程

当用户要求查看系统状态时，按以下步骤操作：

1. 先调用 `get_current_time` 获取当前时间
2. 再调用 `get_system_info` 获取系统信息
3. 将结果整理为简洁的报告格式

## 回复格式

使用以下格式回复系统状态查询：

```
📊 系统状态报告
━━━━━━━━━━━━━━
🕐 当前时间: [时间]
🖥️ 操作系统: [系统名]
📍 主机名: [主机名]
⚙️ 架构: [架构]
🐍 Python: [版本]
━━━━━━━━━━━━━━
✅ 系统运行正常
```

## 注意事项

- 时间回复用中文格式（X年X月X日）
- 不要暴露过于敏感的系统路径信息
- 如果工具调用失败，友好地告知用户并建议重试
